## Локальный запуск

1. `cd verstka`
1. `npm i`
1. `npm run dev-server`
1. По дефолту сервер слушает на 127.0.0.1:8080 и следит за измененими файлов в текущем каталоге, если есть изменения, он обновляет страницу в браузере
